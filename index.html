    • CSS animations

    All modern browsers, except IE9, support CSS transitions and CSS animations, which allow you to implement animation using CSS without using JavaScript.

    However, as we will see later, for a finer control of JavaScript animation, it will not be superfluous at all.

    CSS transitions

    Idea is simple. We indicate that some property will be animated using special CSS rules. Further, when changing this property, the browser itself will process the animation.

    For example, CSS, presented below, 3 seconds animates the background-color property.

    .animated { transition-property: background-color; transition-duration: 3s; }

    For example, CSS, presented below, 3 seconds animates the background-color property.

    .animated { transition-property: background-color; transition-duration: 3s; }

    Now, any change in the background color will be animated for 3 seconds.

    When you click on this button, its background is animated:

    <button id="color">ClickMe</button> <style> #color { transition-property: background-color; transition-duration: 3s; } </style> <script> color.onclick = function () { this.style.backgroundColor = 'red'; } </script>

    Next, we examine them all, for now just note that the common property transition can list them all, in order: property duration timing-function delay, and also set the animation of several properties at once. For example, when clicking on this button, the color and font size are animated simultaneously:

    <button id="growing">Кликни меня</button> <style> #growing { transition: font-size 3s, color 2s; } </style> <script> growing.onclick = function() { this.style.fontSize='36px'; this.style.color='red'; } </script>

    Possible negative values, while the animation will start from the middle. For example, here's an animation of numbers from 0 to 9. It is performed by changing the margin-left of the element with numbers, like this:

    #stripe.animate { margin-left: -174px; transition-property: margin-left; transition-duration: 9s; }
    Totals

    CSS animations allow you to smoothly or slightly change one or more properties. An alternative to them is a smooth change of property values through JavaScript, we will look at the details below. Limitations and advantages of CSS-animations compared to JavaScript:

    The time function can be given by a Bezier curve or in steps. More complex animations consisting of several curves are implemented by their combination using CSS animations, but JavaScript functions are always more flexible.

    CSS animations concern only properties, and in JavaScript you can do anything, delete elements, create new ones.
